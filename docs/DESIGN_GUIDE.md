# DESIGN_GUIDE — 카드 & UI 디자인 가이드

> 목표: "포켓몬 카드처럼 보이게 베끼기"가 **아니라**, 트레이딩 카드 장르의 익숙함만 자체 디자인/세계관으로 구현. IP 규칙은 [`IP_RISK_GUIDE.md`](./IP_RISK_GUIDE.md) 참조 — 본 문서와 항상 함께 본다.

## 1. 디자인 톤

AI · 네온 · 홀로그램 · 미래형 · 아케이드 · 리더보드/이벤트 · 디지털 컬렉션.
키워드: "디지털 등급 배틀 카드". 종이 카드 모사 X, **디지털 네이티브** 룩.

## 2. 카드 레이아웃 (공식 포켓몬과 의도적으로 다르게)

```text
┌───────────────────────────────┐
│ [이름]            [등급 배지]  │  상단: 이름 + 등급(SSS 등)
│              [SCORE/PWR]       │  우상단: 점수/PWR (HP 위치와 다르게)
│ ┌───────────────────────────┐ │
│ │                           │ │
│ │   사용자 사진 (1컷)        │ │  중앙: 메인 이미지 박스
│ │                           │ │
│ └───────────────────────────┘ │
│ ▌속성  AI 분석 요약            │  좌측 세로바: 속성 아이콘 / 중하단: 분석요약
│ ── 대표 스킬 ───────────────── │  하단1: 대표 스킬 (단일, GX식 대형 표기 금지)
│ #카드ID   주간랭킹 7위         │  하단2: 카드ID / 이벤트 랭킹
│            [서비스 로고]       │  맨아래: 자체 브랜드 로고
└───────────────────────────────┘
```

**금지(레이아웃)**: HP/속성/이름을 공식 카드와 동일 위치 배치, 약점/저항력/후퇴비용 3단 하단 구조, GX/EX식 대형 하단 스킬 표기, 은색 테두리+기술박스+타입색 조합 복제, 하단 copyright/카드번호 레이아웃 복제. 세로형·이미지박스·등급·속성·스킬·홀로 효과·컬렉션번호는 장르 문법으로 허용.

## 3. 용어 (대체 표기 — 강제)

| 금지 | 사용 |
|---|---|
| HP | **PWR / SCORE / AURA / CP** |
| 특성 | Core Skill / Passive / Trait |
| 기술 | Move / Skill / Signature |
| 약점 | Risk |
| 저항력 | Shield |
| GX/EX/VMAX/VSTAR | Apex / Prism / Burst / Overdrive |
| 진화 | Stage / Form / Level |
| 에너지 | Core / Element / Aura |

## 4. 등급 배지

| 등급 | 희귀도 | 톤(예시) |
|---|---|---|
| SSS+ | Mythic | 무지개 홀로 + 파티클 |
| SSS | Legendary | 골드 홀로 |
| SS | Ultra Rare | 프리즘 실버 |
| S | Super Rare | 네온 글로우 |
| A | Rare | 컬러 글로우 |
| B | Uncommon | 매트 컬러 |
| C | Common | 모노톤 |

등급 배지는 자체 형상(육각/쉴드 등) 사용. 포켓몬 레어도 아이콘(별/원 등) 모사 금지.

## 5. 속성 아이콘 (자체 시각 언어)

원형 안 단순 심볼(=포켓몬 에너지 유사) **금지**. 자체 문양 사용:

```text
Aqua=물방울+파형  Flame=삼각불꽃+균열  Terra=육각결정  Storm=회오리전류(번개X)
Neon=빛나는 렌즈  Shadow=초승달 그림자  Metal=금속결정  Spirit=오라 링
Solar=태양 파편   Lunar=달 조각
```

## 6. 프레임 세트

Prism / Neon / Aqua / Shadow / Solar / Lunar / Metal / Event Limited.
MVP는 자체 디자인 프레임 5~10종. 속성·등급에 따라 자동 추천, 사용자는 스타일만 선택(수치 무관).

## 7. 컬러 / 타이포 / 모션

- 컬러: 다크 베이스 + 네온 액센트(속성별). 홀로는 그라디언트/노이즈 오버레이로 표현.
- 타이포: 기하학적 산세리프 + 수치용 모노스페이스. 포켓몬 폰트/로고 유사체 금지.
- 모션: 분석 중 화면 = 스캔 라인 + AI 게이지 + 카드 프레임 로딩. 결과 공개 = 등급 리빌(홀로 스윕).

## 8. 공유 이미지 규격

| 용도 | 규격 |
|---|---|
| 앱 내 카드 | 고해상도 PNG |
| 정사각(기본) | 1080×1080 |
| 인스타 스토리 | 1080×1920 |
| 인스타 피드 | 1080×1350 |
| OG(카카오/페북) | 1200×630 |

스토리 구성: 상단 등급 / 중앙 카드 / 하단 "랭킹 7위 · 나도 만들기". 전 이미지에 워터마크+서비스 URL.

## 9. 핵심 화면

온보딩 · 홈(카드 만들기/진행 이벤트/주간 TOP3/내 카드/리더보드) · 촬영·업로드(가이드/크롭/회전) · 분석 중 · 카드 결과(수정 버튼 없음) · 리더보드(일/주/월/이벤트/전체 탭) · 카드 상세 랜딩 · 관리자.

## 10. 접근성/원칙

명도 대비 준수, 등급/속성은 색+텍스트 병기(색맹 대응), 모션 줄이기 옵션. **결과 화면에 점수/등급/능력치 수정 UI 절대 미노출.**
