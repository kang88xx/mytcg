# IP_RISK_GUIDE — IP 리스크 관리 가이드

> 원칙: 포켓몬 IP를 직접 쓰지 않는다. 트레이딩 카드 **장르 문법**까지만 허용하고, 외형·명칭·아이콘·세계관은 자체 제작한다. 본 문서는 [`DESIGN_GUIDE.md`](./DESIGN_GUIDE.md)와 함께 강제 적용된다.

## 1. 절대 금지 (명칭/요소)

```text
포켓몬 / Pokémon / Pokemon / Poke / 포켓 / 몬스터볼 / Pokéball
GX / EX / VMAX / VSTAR / 1진화 / 2진화
포켓몬 에너지 아이콘과 유사한 원형 속성 아이콘
공식 카드와 유사한 HP 위치 / 타입 아이콘 배치
약점 / 저항력 / 후퇴비용 3단 구조 복제
공식 카드 하단 copyright / 카드번호 레이아웃 복제
은색 테두리 + 기술 박스 + 타입 색상 조합 복제
앱 이름·설명에 "포켓몬 카드 만들기"
마케팅에서 "포켓몬 스타일" 표현
```

## 2. 대체 용어 (강제)

| 금지 | 대체 |
|---|---|
| HP | PWR / SCORE / AURA / CP |
| 특성 | Core Skill / Passive / Trait |
| 기술 | Move / Skill / Signature |
| 약점 | Risk |
| 저항력 | Shield |
| GX | Apex / Prism / Burst / Overdrive |
| 진화 | Stage / Form / Level |
| 에너지 | Core / Element / Aura |

## 3. 레이아웃 차별화 규칙

- HP/속성/이름 배치를 공식 카드와 동일하게 두지 않는다.
- 약점/저항력/후퇴비용 3단 하단 구조를 만들지 않는다.
- GX/EX식 대형 하단 스킬명 표현을 피한다.
- 속성 아이콘은 원형+단순심볼이 아니라 자체 문양(파형/균열/결정 등)으로.
- 자체 카드 세계관·명칭 체계를 둔다.

## 4. 출시 전 검수 체크리스트

```text
□ 포켓몬 카드와 나란히 놓아도 같은 카드로 안 보인다
□ 포켓몬 로고/명칭/몬스터볼/GX/EX/에너지 아이콘 없음
□ 앱스토어 검색어/메타데이터에 Pokemon/Poke/포켓몬 없음
□ 카드 정보 위치가 공식 카드와 다르다
□ 유사성은 "장르 문법"까지만
□ 외형/명칭/아이콘/세계관은 자체 제작
□ 마케팅 문구에 "포켓몬 스타일/포켓몬 카드 만들기" 없음
□ 앱 소개는 "AI 트레이딩 카드 / AI 배틀 카드 / AI 카드 랭킹"
```

- 사용: "AI 트레이딩 카드", "AI 배틀 카드", "AI 카드 랭킹", "내 사진이 AI 카드로".
- 금지: 포켓몬/Pokemon/Poke/포켓/몬스터 카드, "포켓몬 스타일".
- 브랜드명 후보: CardRank AI / Aura Card / Prism Rank / 오라카드 / 프리즘카드 (브리핑 §22).

## 7. 이벤트/플랫폼 비후원 고지

상품 이벤트는 **Apple/Google/Meta/Kakao와 무관**함을 약관·앱 내에 명시. 참가비 없음, 결제로 순위 상승 불가, 점수 기반 랭킹.
